# find-wifi-front

Find Wi-Fi フロントエンド [React]

## 技術要素

- TypeScript `3.6.x`
- React `16.9.x`

## ローカル開発環境構築手順

- リポジトリのクローン

```
$ git clone git@github.com:KotaTanaka/find-wifi-front.git
```

- パッケージのインストール

```
$ cd find-wifi-front
$ yarn
```

- アプリケーションの起動

```
$ yarn start
```

→ http://localhost:3010 でアプリケーションが起動します。
